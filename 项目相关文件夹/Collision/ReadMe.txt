
This directory contains the Open Inventor demo programs for Collision features:

Dart            - Move and send arrow (with pick mode and space bar).

ObjectMoving    - Use a manipulator selected in the menu (jack) and translate
                  the cube, if a collison is detected -> switch color (red).

CollisionViewer - Move into a scene without crossing the wall!

Intersections   - Select different part and press space bar
